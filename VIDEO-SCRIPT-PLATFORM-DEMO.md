## üé• Recording Tips

### Technical Setup:
1. **Screen Resolution:** 1920x1080 (Full HD)
2. **Recording Software:** OBS Studio, Loom, or Camtasia
3. **Frame Rate:** 30 fps minimum
4. **Audio:** Clear microphone, no background noise
5. **Cursor:** Enable cursor highlighting

### Visual Guidelines:
1. **Smooth Navigation:** Move slowly between screens
2. **Pause on Key Info:** Hold on important data for 2-3 seconds
3. **Highlight Actions:** Use cursor to point at key elements
4. **Clean Browser:** Close unnecessary tabs, use incognito mode
5. **Test Data:** Use realistic but clear test data

### Pacing:
- **Total Length:** 10-11 minutes
- **Intro:** 1:30 (set context)
- **Farmer Journey:** 3:30 (detailed walkthrough)
- **Investor Journey:** 3:30 (detailed walkthrough)
- **Platform Features:** 2:00 (showcase capabilities)
- **Conclusion:** 1:00 (call to action)

### Voice Over:
- **Tone:** Professional but warm and engaging
- **Pace:** Moderate - not too fast, clear enunciation
- **Emphasis:** Highlight key numbers and benefits
- **Pauses:** Brief pauses between sections
